## ğŸ› ï¸ Installation & Local Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local development server (Python or Node.js)

### Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/syedtanvircodex/portfolio.git
   cd portfolio
   ```

2. **Start a Local Server**

   **Using Python 3:**
   ```bash
   python -m http.server 8000
   ```

   **Using Python 2:**
   ```bash
   python -m SimpleHTTPServer 8000
   ```

   **Using Node.js (with http-server):**
   ```bash
   npx http-server -p 8000
   ```

3. **Open in Browser**
   - Navigate to `http://localhost:8000`
   - Open `index.html` to view the portfolio

### Why a Local Server?
- Bypasses CORS restrictions for API calls
- Enables proper module imports and fetch operations
- Ensures accurate rendering of assets and styles
